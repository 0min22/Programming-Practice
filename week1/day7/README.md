# Log Analyzer

간단한 파이썬 로그 분석 프로그램입니다.

## 기능

* `log.txt` 파일을 읽습니다.
* 전체 로그 줄 수를 출력합니다.
* `LOGIN`, `ERROR`, `FAIL` 개수를 각각 출력합니다.
* `FAIL`이 3회 이상이면 **위험 사용자**를 출력합니다.
* `ERROR`가 3회 이상이면 **경고**를 출력합니다.

## 사용한 개념

* 파일 읽기 (`with open`)
* 반복문 (`for`)
* 조건문 (`if`, `elif`)
* 딕셔너리
* 문자열 처리 (`strip`)

## 실행 방법

1. `log.txt` 파일을 같은 폴더에 준비합니다.
2. 파이썬 파일을 실행합니다.

예시:

```bash
python log_analyzer.py
```

## 예시 로그 파일

```text
LOGIN
ERROR
LOGIN
FAIL
ERROR
LOGIN
FAIL
```

## 예시 출력

```text
전체 로그 수: 7
LOGIN : 3
ERROR : 2
FAIL : 2
```

## 느낀 점

이 프로젝트를 통해 파일에서 데이터를 읽고, 조건에 따라 필요한 정보만 세어서 출력하는 방법을 연습할 수 있었습니다. 또한 단순한 문법 연습을 넘어서 실제로 동작하는 작은 프로그램을 만들어보며 파이썬의 기본 구조를 익힐 수 있었습니다.
